Major Pieces of the Web
• Browsers make requests via the internet to servers
• Browsers- a program that runs on your computer to display files found on the web
• Internet- the world's largest computer network
• Servers- computers that host the files that make up the web
HTTP- the main protocol of the web; the internet uses HTTP to communicate with you, your browser, and servers
